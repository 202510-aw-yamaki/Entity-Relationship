# 判定ルール固定書

## 目的
本書は STEP1 / STEP2 / STEP3 の判定条件と、日本語メッセージの出し方を固定する。

## 共通方針
- 判定結果は必ず日本語の文章で表示する
- エラーコードだけを出さない
- 問題がない場合も「問題なし」と分かる文章を出す
- 警告がある状態で次 STEP へ進むときは確認ポップアップを出す
- 問題のある箱や線は可能な限り特定する
- ダミーの OK 判定は出さない

## STEP1 判定条件

### 目的
- 顧客要望や大まかな関係性を可視化する
- 主要カテゴリが揃っているか確認する
- 箱が独立していないか確認する
- 未入力や未設定を確認する

### 最低限の判定項目
- 主要カテゴリ不足
- カテゴリ未設定
- タイトル未入力
- 独立箱
- 線が全くない

### 判定内容の固定
- 主要カテゴリ不足は、必要カテゴリの箱が 1 つもない場合に指摘する
- カテゴリ未設定は、箱に `category` が入っていない場合に指摘する
- タイトル未入力は、箱に `title` が入っていない場合に指摘する
- 独立箱は、どの線にもつながっていない箱を指摘する
- 線が全くないは、線一覧が空のときに指摘する

## STEP2 判定条件

### 目的
- フロント / バック / 保存 / 参照の関係を確認する
- 特に保存先と参照先の関係が曖昧でないか確認する

### 最重要の判定項目
- 保存データがあるのに保存線がない
- 参照データがあるのに参照線がない
- 保存データがどこから来るか不明
- 参照データがどこで使われるか不明
- 画面表示に必要な参照元が不明
- ユーザー入力の行き先が不明

### 判定内容の固定
- 保存データカテゴリの箱が存在する場合、少なくとも 1 本は `保存` 関係の線が必要
- 参照データカテゴリの箱が存在する場合、少なくとも 1 本は `参照` 関係の線が必要
- 保存データの箱に接続する `保存` 線がない場合は、保存元不明として指摘する
- 参照データの箱に接続する `参照` 線がない場合は、利用先不明として指摘する
- 画面表示カテゴリの箱があるのに、参照データとの関係が確認できない場合は、参照元不明として指摘する
- ユーザー入力カテゴリの箱があるのに、接続先が確認できない場合は、行き先不明として指摘する

STEP2 では、保存先と参照先の関係を最優先で厳しく確認する。

## STEP3 判定条件

### 目的
- 顧客にも作成者にも分かる粒度まで説明を落とす
- 箱や線の意味が説明できるようにする

### 最低限の判定項目
- 箱に顧客向け注釈がない
- 線に説明がない
- タイトルはあるが意味合い説明が薄い
- 顧客要望カテゴリの説明不足
- 箱や線の意味が後で見返しても分かりにくい

### 判定内容の固定
- `customerNote` が空の箱は指摘する
- `label` が空の線は指摘する
- `title` はあるが `meaning` が空、または極端に薄い場合は指摘対象とする
- 顧客要望カテゴリの箱で `meaning` または `customerNote` が不足している場合は優先して指摘する
- 箱や線の名称だけでは意味が分からない場合、説明不足として扱う

## 不足メッセージ例

### STEP1 向け
- このカテゴリの箱が現在ありません: 保存データ
- この箱はカテゴリがまだ選ばれていません: 顧客一覧画面
- この箱はタイトルが未入力です: node-1
- この箱は独立しています: 顧客一覧画面
- まだ線が 1 本もありません。関係が分かるようにつないでください

### STEP2 向け
- 保存先はありますが、どこから保存されるかが不明です: 顧客マスタ
- 参照先はありますが、どこで使われるかが不明です: 商品一覧データ
- この画面表示に必要な参照元が分かりません: 顧客一覧画面
- このユーザー入力の行き先が分かりません: 検索条件入力
- 保存データがありますが、保存の線がまだありません

### STEP3 向け
- この箱には顧客向けの補足説明がありません: 見積依頼
- この線はつながっていますが、何の関係か説明がありません
- この箱はタイトルだけでは意味が分かりにくいため、意味合いの説明を足してください: 一覧
- この顧客要望は説明が不足しています: 見積依頼を受け付けたい
- 後で見返したときに分かりにくい線があります: edge-3

## 問題なしメッセージ例
- STEP1 の確認では、現時点で大きな不足は見つかっていません
- STEP2 の確認では、保存先と参照先のつながりに大きな問題は見つかっていません
- STEP3 の確認では、説明不足として強く気になる点は見つかっていません
