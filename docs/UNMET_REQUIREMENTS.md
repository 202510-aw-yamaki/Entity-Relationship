## 未達の項目

### UI / 操作性メモ

- 箱を選択した状態で `Ctrl+C` でコピーし、`Ctrl+V` で複製できるようにしたい
- 箱を置いて作業する図エリアが狭いため、作業領域を拡張したい
- 各 STEP 判定はサイドバーを開かなくても実行できるようにしたい
- 判定ショートカット案として `Ctrl+S1` で STEP1、`Ctrl+S2` で STEP2、`Ctrl+S3` で STEP3 を実行できるようにしたい
- 箱の要件へ追加入力しようとすると 1 文字でフォーカスが外れるため、連続入力できるように修正したい
